> "我们为黄金而来，却惊醒了那沉睡之物，大地与天空悲鸣不息，月亮泣血不止；自那晚后，世界就变了。天上掉下来的不是星星，是诅咒…也是恩赐。它们对着所有能听见的东西低语，许诺给你力量，然后扭曲你的意志。
> 
> 大多数人都没扛住，变成了怪物。而我们…我们这些'幸运儿'，只不过是把疯病暂时锁在了脑子里，用怪物的力量去猎杀怪物。
>

### 核心世界观：萨特之镐

#### 起源
1850年1月，西部淘金热已持续两年，越来越多的人为了黄金涌入西部这片土地。浅层的地脉基本已被挖掘殆尽，可人们心中的贪婪仍在滋长，矿井越来越深，直到有一天，一名淘金客的镐头敲碎了一层古老的岩壳，惊醒了沉睡于古老土地之下的不可名状之物。

随即而来的就是那场史无前例的月蚀，地底之物不断吸收着从天而降的白色结晶。这些结晶如同有生命的种子，在地表扎根，并持续散发着充满诱惑与疯狂的低语，渴望与生命融合，将生命推向一个更高、但也更非人的进化阶段。

#### 畸变体的真相：失败的同化者
绝大多数生物的心智无法承受低语的冲刷。他们的肉体在结晶能量的作用下急速变异、膨胀，但意志却彻底崩溃，沦为只受本能和低语驱使的怪物——畸变体。它们既是这场灾变的受害者，也是结晶能量在世界上最为活跃的载体。击杀它们，正是获取已与生物质融合的、更稳定结晶的唯一方式。

#### 猎魔人与女巫：危险的幸运儿
猎魔人和女巫，是极少数的幸运儿。他们接触了结晶，听到了低语，在疯狂的边缘稳住了心神，并成功地驯服了部分能量为己所用。他们获得了超乎常人的力量，但也永远与低语为伴。每一次使用力量，都像是在悬崖边漫步，有再次坠入畸变深渊的风险。他们猎杀畸变体，既是为了收集更多的结晶来强化自身，也是为了在这崩坏的世界中寻求一线秩序。

## 阵营设定

### 1. 边境猎魔人
**背景**：主要由前平克顿侦探、退伍骑兵、法外之徒组成。他们大多是实用主义者。认为对结晶力量的使用必须加以严格的限制，将结晶能量通过复杂的工艺烙印在武器和工具上。他们信任经过改造的杠杆步枪胜过信任自身不稳定的意志。

**特点**：外部驾驭。力量来自于结晶科技。技能围绕强化武器展开。

### 2. 女巫会
**背景**：传承自欧洲古老秘法社团，为调查这场灾难的源头而远渡重洋。她们将月色结晶视为强大的施法媒介，试图终结这场灾难。

**特点**：能驱动结晶能量，施展范围性的法术、瞬移和防护结界，身体相对脆弱。

### 3. 暗影行会（潜在第三阵营/中立方）
**背景**：一个极端的神秘教派。他们相信畸变并非失败，而是通往真我的必经之路。他们主动追求彻底放开意志，与低语合一，成为更完美的畸变体。他们猎杀畸变体，是为了汲取同胞的经验，并试图找到那条可控的、指向终极进化的路径。

**特点**：拥抱疯狂。拥有独特的腐化值机制，通过攻击或特殊行为积累，积满后可短暂变身为半畸变体，获得巨大力量，但结束后陷入虚弱。

## 游戏核心玩法（PVPVE）

每场对局都是一次高风险高回报的狩猎行动。

### 进入战场
最多3名玩家组成小队，选择阵营，携带自定义的杠杆步枪、技能和补给，空降至一张西部地图。

### 猎杀与搜集
地图上散布着低阶畸变体。击杀它们有概率掉落少量"结晶碎片"。同时，地图上会刷新数个腐化巢穴，清除巢穴核心能获得大量结晶，但会引发巢穴的疯狂反扑，巨大的动静极易暴露你的位置。

### 追踪巨兽
地图某处会随机刷新一只强大的Boss级畸变体。击败它能获得最丰厚的"月色结晶"奖励以及稀有武器蓝图。但击杀巨兽造成的能量波动会扩散至全地图被其他小队感知，你将成为所有其他玩家小队的猎物。

### 生存与撤离
当收集到足够结晶或弹尽粮绝，局势不利时，玩家可以前往地图上的撤离点。游戏对局结束前，任何玩家队伍都能撤离。但要注意，敌人可能会在最后一刻伏击你，夺走你的一切。

### 永久的风险
若在对局中阵亡，你的角色以及所携带的装备和本轮收集的所有结晶将全部丢失。唯有成功撤离，才能用结晶升级武器、学习新技能，为下一场更危险的狩猎做准备。

### 理智值系统
每名角色有一个次要资源条。多次使用技能、停留在地图危险区域都会累积侵蚀"。侵蚀度过高会出现幻听幻视，屏幕特效扭曲，甚至吸引畸变体主动攻击。使用特殊道具或返回安全区才能降低。

## 开发要求

### 系统要求
1. Unreal Engine 5.3+
2. Visual Studio 2022 (Windows)
3. Git for version control

### 硬件要求
- Windows 10/11 (64-bit)
- DirectX 11/12 compatible graphics card
- 16GB+ RAM recommended
- 50GB+ available disk space

## 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/1661226275/Westword.git
```

2. 使用Unreal Engine打开 `Westword.uproject`

3. 编译项目
```bash
# 在项目根目录下
"C:\Program Files\Epic Games\UE_5.3\Engine\Build\BatchFiles\Build.bat" WestwordEditor Win64 Development -Project="Westword.uproject" -WaitMutex -FromMsBuild
```

## 开发架构

本项目基于虚幻引擎5构建，采用模块化设计：

- **核心系统**：基础角色控制器和游戏模式
- **变体系统**：不同游戏模式的独立实现
- **武器系统**：可扩展的武器框架
- **AI系统**：智能NPC和敌人行为
